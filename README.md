** Gaussian Mixture Model (GMM) Using EM Algorithm**

This project implements a Gaussian Mixture Model (GMM) from scratch using the Expectation-Maximization (EM) algorithm in R. The model generates synthetic data consisting of two Gaussian clusters and performs probabilistic clustering without using external machine learning libraries.

The implementation includes dataset generation, parameter initialization, Gaussian density calculation, Expectation Step (E-Step), Maximization Step (M-Step), log-likelihood computation, convergence checking, and final cluster assignment. The project also visualizes the generated dataset, clustered output, and log-likelihood progression.

 **Features**

* Generate synthetic Gaussian datasets
* Implement GMM from scratch
* Expectation-Maximization (EM) algorithm
* Soft and hard clustering
* Log-likelihood convergence analysis
* Cluster visualization and output generation
* Supports large datasets (up to 20,000+ records)

 **Technologies Used**

* R Programming Language
* Matrix Operations
* Statistical Modeling
* Data Visualization

** Output**

* Synthetic Dataset (CSV)
* Cluster Assignment Results
* Dataset Visualization
* Final Cluster Visualization
* Log-Likelihood Convergence Graph

This project demonstrates how Gaussian Mixture Models can be used for unsupervised learning, density estimation, and clustering analysis using probabilistic methods.
